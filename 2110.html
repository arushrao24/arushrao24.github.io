Hobgobbler Robot

Overview

  • Robot for Georgia Tech's ME 2110 competition
  • Multi-funtional robot following the "How To Train Your Dragon" theme
  • Designed and built from scratch apart from mechatronics components

My Role

  • Designed and modeled 30+ components in SolidWorks contributing to multiple subsystems (ie. drivetrain)
  • Used 3D printing to rapidly prototype parts and refine designs
  • Laser cut a plywood frame for the robot
  • Assembled and refined robot using various woodworking tools (jigsaw, handsaw, drill, etc)
  • Integrated mechatronics components like motors, solenoids, and pneumatic actuators into robot
  • Wrote Arduino code to precisely control the robot's movements and subsystem functionality
  • Created a bill of materials, specification chart, function tree, and morphological chart
  • Wrote multiple formal techanical reports and presenations to share designs, progress, and results
